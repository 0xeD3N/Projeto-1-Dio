História da Web

Nesta primeira etapa do inicio do curso, aprendemos como foi o surgimento das primeiras maquinas até o que a gente conhece hoje em dia.

Nela vimos o surgimento das primeiras linguagens de programação e como se iniciou o que hoje conhecemos como Web.

O que são Clients?

Client, basicamente é a maquina, ou meio, que o usuario utiliza para obter alguma informação.

Exemplo:

  • Imagens
  • Textos
  • Mídias

Os Clients fazem a comunicação com a internet, que por sua vez faz a comunicação com um servidor que possui a informação.

O que são Servers?

Por sua vez, os Servers são uma maquina robusta que armazena dados que possam ser acessados pelos Clients.

Um server pode ter varias funcões e varios tipos.

Exemplo:

  • Servidor de Arquivos
  • Servidor Web
  • Servidor de Aplicativos
  • Servidor de APIs
  • Servidor de Gerenciamento
  • Servidor de e-mails
  • Servidor Firewall

Os Servers recebem a comunicação da internet e a retornam com os dados e informações requeridas pela mesma.

Estrutura básica do HTML

A estrutura básica do HTML é a seguinte:

Começamos utilizando a tag "html".

Dentro do HTML, possui as tags:

  • head
  • body
A tag "head" contém as informações que serão utilizadas para pré-carregar as informações do documento.
  • title
  • meta
  • link
A tag "body" contém as informações que serão utilizadas para acessar as informações do documento.
  • h1
  • h2
  • p
Utilizando destas tags, conseguimos construir um site utilizando o HTML.

Falando sobre tags

A tag "h1", "h2", "p" e "a" são as tags mais utilizadas no HTML.

Elas são utilizadas para adicionar um texto na sua pagina, segindo uma hierarquia.

com "h1" sendo o título;

"h2" o sub-título;

"p" o texto;

"a" o link.

Atributos das Tags

Cada tag possui seus diferentes atributos, sendo eles universais ou próprio de cada tag.

Atributos universais

  • id
  • class
  • style

Atributos próprios

  • o href da tag a
  • o src da tag img

Esses atributos podem adicionar propriedades especificas para suas determinadas tags, como:

  • class
  • id
  • style
  • name

Listas Ordenadas e Não Ordenadas

Listas Ordenadas

Listas ordenadas são utilizadas para organizar os itens enumerados dentro de uma lista.

Utiliza-se a tag "ol" para declarar que uma lista segue uma ordem numérica.

Dentro da tag "ol" deve ser utilizado outra tag, denominada "li".

Listas Não Ordenadas

Listas não ordenadas são utilizadas para organizar os itens não enumerados dentro de uma lista.

Utiliza-se a tag "ul" para declarar que uma lista não segue uma ordem numérica.

Dentro da tag "ul" deve ser utilizado outra tag, denominada "li".

Links

Vimos que a tag "a" é utilizada para adicionar um link na sua pagina.

A tag "a" possui dois atributos:

  • href
  • target
Sendo "href" utilizado para apontar aonde está "ancorado" a informação e "target" como sera mostrado essa informação.
